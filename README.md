# Numerical Methods (Java ☕)

A Java-based scientific computing application with a GU  for numerical methods, including integration, differentiation, solving nonlinear equations, and related applications.  
## 🛠️ Libraries Used
Expression Evaluation: [Exp4j](https://github.com/fasseg/exp4j)

GUI: Swing + [FlatLaf](https://github.com/JFormDesigner/FlatLaf)

Graphs: [JFreeChart](https://github.com/jfree/jfreechart)

# 📖 Contents

## 📊 1. Numerical Integration

1.  Riemann Sums 
2.  Trapezoidal Rule 
3.  Simpson 1/3 Rule 
4.  Romberg Integration
5.  Gauss Quadrature

## 📈 2. Numerical Differentiation

1.  Forward Differentiation
2.  Backward Differentiation
3.  Central Differentiation

## 📌 3. Root Finding

1.  Bisection Method
2.  Newton-Raphson Method
3.  Secant Method

## 📦 4. Applications

1.  Average 
2.  Area
3.  Volume
4.  Arc Length
5.  Surface Area
